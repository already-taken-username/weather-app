# Weather forecast app

### Based on [React.js](https://https://reactjs.org/) library and [openweathermap.org](https://openweathermap.org) free api, fetch data and display it for users.

## How to use

Installation
```
npm install
```
start the server
```
node .
```
visit in your browser
```
localhost:8080
```
or
```
127.0.0.1:8080
```

## Features

* changable localization based on 
  - geolocalization
  - user typed text
* current weather
  - temperature
  - wind speed and direction
  - pressure
  - humidity
  - cloudiness
  - visibility
* upcoming forecast for next 4-5 days
  - temperature
  - wind speed
  - pressure
  - humidity
  - cloudiness
  - visibility
* forecast timeline data for every 3 hours
* dynamic time bar shows
  - sunrise sunset and local time for chosen localization
  - upcoming sunrise or sunset count down (appear 60 minutes before)
* dynamic theme color based on local time

## Used technologies:

* React.js
* Express.js
